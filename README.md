<EN> PortalX 
• In small-medium sized businesses, it is difficult to produce and operate an interface for internal communication. <br>
PortalX was designed to meet this need and contribute to business efficiency. <br>
• A simplified management interface has been added to perform LDAP/Active Directory operations within PortalX. <br>
• PortalX have pages to add/edit/debit Fixtures records. <br>
• There are manageable news, announcement pages, phone book, food menus, shuttle lists, fixtures and communication boards. It can serve as an environment where institutional documents can be shared. <br>
• Backup and restore processes are automated. <br>
• The PortalX is designed to be usable immediately after installation. It requires minimum support during the installation and commissioning phases. <br>
• It is a completely open source software. Since it is coded in a common language, it is suitable for modifications later. <br>
• PortalX is licensed under an open source license (MIT) and is completely free. For this, please refer to the 3. LICENSE.txt document in the code. Installing It will be installed by following the order described in the install.txt document. <br>
